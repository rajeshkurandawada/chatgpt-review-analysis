# ChatGPT Reviews Analysis

## Project Overview
This project analyzes user reviews of ChatGPT to understand sentiment and rating distribution.

## Objectives
- Analyze rating distribution
- Perform sentiment classification
- Identify user satisfaction patterns

## Dataset
The dataset contains:
- Review Id
- Review
- Ratings (0-5 scale)
- Review Date

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Conclusion
Most users have given high ratings (4-5), indicating positive sentiment. A smaller percentage of users expressed neutral or negative feedback.
